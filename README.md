# Strooke

## Contexto
Proyecto: Desarrollo de Plataforma Web para Venta y Distribución de Gorras
Descripción General
El proyecto consiste en el desarrollo de una plataforma web completa para la comercialización y distribución de gorras urbanas y deportivas.
El sistema permitirá la gestión integral de usuarios, productos, pedidos y envíos, incorporando funcionalidades que mejoren la experiencia del cliente y optimicen la administración del emprendimiento.
